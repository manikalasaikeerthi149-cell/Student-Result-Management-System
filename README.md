# 🎓 Student Result Management System

A modern and responsive **Student Result Management System** built using **HTML, CSS, and JavaScript**. This project provides an interactive interface for managing student results, calculating grades automatically, and displaying useful statistics through a clean and attractive dashboard.

## ✨ Features

- ➕ Add student details
- 📊 Automatic total marks calculation
- 📈 Automatic average marks calculation
- 🏆 Automatic grade generation
- 🔍 Search students by name or roll number
- 🗑️ Delete student records
- 📋 Responsive results table
- 📌 Dashboard statistics
  - Total Students
  - Average Marks
  - Highest Marks
- 🎨 Modern Glassmorphism UI
- 📱 Fully Responsive Design

---

## 📋 How It Works

1. Enter the student's name and roll number.
2. Enter marks for Math, Science, and English.
3. Click **Add Result**.
4. The system automatically:
   - Calculates total marks.
   - Calculates average marks.
   - Assigns a grade.
   - Updates dashboard statistics.
5. Search students using the search box.
6. Delete records whenever required.
